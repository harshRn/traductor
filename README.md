# Traductor

Read EPUB and PDF in any of nine languages. Tap a word to translate.
Save it. Watch your dictionary grow into the words you actually look up.

## Acknowledgements

Traductor is inspired by [LUTE (Learning Using Texts)](https://github.com/LuteOrg/lute-v3)
by Jeff Zohrab — an open-source self-hosted web app that pioneered the
"learn a language by reading and tapping words" workflow on desktop.
Traductor is an independent iOS reimagining of the same idea, written
from scratch in Swift; no LUTE source code is used. LUTE is MIT-licensed.
